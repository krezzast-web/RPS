## 🎨 Step 3: Create the 3D Hand Viewer Component
Create a new component at `src/components/Hand3D.jsx` to load and display the models based on the game state.

```javascript
import React, { Suspense } from 'react';
import { Canvas } from '@react-three/fiber';
import { useGLTF, Center, OrbitControls } from '@react-three/drei';

// Individual model loaders
function Model({ path }) {
  const { scene } = useGLTF(path);
  return <primitive object={scene} scale={1.5} />;
}

export default function Hand3D({ selection, isOpponent }) {
  // Determine which model path to load
  let modelPath = '/src/assets/3d/hand_rock.glb'; // Default fist for shaking/waiting
  
  if (selection === 'P') {
    modelPath = '/src/assets/3d/hand_paper.glb';
  } else if (selection === 'S') {
    modelPath = '/src/assets/3d/hand_scissors.glb';
  }

  return (
    <div style={{ width: '100px', height: '100px' }}>
      <Canvas camera={{ position: [0, 0, 4], fov: 45 }}>
        <ambientLight intensity={0.7} />
        <directionalLight position={[5, 5, 5]} intensity={1} />
        <Suspense fallback={null}>
          <Center>
            {/* Mirror/flip opponent's hand horizontally on the Y-axis */}
            <group rotation={[0, isOpponent ? Math.PI : 0, 0]}>
              <Model path={modelPath} />
            </group>
          </Center>
        </Suspense>
      </Canvas>
    </div>
  );
}
```

## 💫 Step 5: Animating the Shaking Sequence
Since the shaking animation is currently run by CSS keyframes in [index.css](file:///c:/Users/trojan/Desktop/web/RPS/src/index.css#L1284-L1301) on `.hand-visual-container`, the 3D Canvas will inherit this shake animation automatically. 

If you want more realistic 3D movement (e.g. true 3D rotation shifts instead of flat up-and-down CSS motion), you can implement it using the R3F `useFrame` hook directly inside the component:

```javascript
import { useFrame } from '@react-three/fiber';
import { useRef } from 'react';

// Inside your Model or Group component:
const groupRef = useRef();
useFrame((state) => {
  if (isShaking) {
    // Generate a smooth sinusoidal shake on the Y axis
    groupRef.current.position.y = Math.sin(state.clock.getElapsedTime() * 10) * 0.2;
    groupRef.current.rotation.z = Math.sin(state.clock.getElapsedTime() * 10) * 0.1;
  }
});
```
