Every prompt has been updated to specify that the hand wears a **dark grey/black glove with the Rpsroom logo** (a graphic symbol composed of a semi-circle, a circle, and a triangle) on the back of the hand.

## 💡 Tips for Best Results in Meshy AI:
1. **Target Polycount:** Select the **Low Poly** option in Meshy if you plan to import these directly into the web interface as three.js / GLTF assets to keep loading times fast.
2. **Texture Style:** Choose **Realistic** or **Stylized** matching the style group you selected above.
3. **Symmetry/Pose:** Ensure you select a neutral standing posture or layout in Meshy if it asks for rigging/initial layout.
